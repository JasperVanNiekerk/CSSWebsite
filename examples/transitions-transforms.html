HTML

  Hover Me!
</div>

<div class="transition-demo">
  <h4>Transition Card</h4>
  <p>This card demonstrates smooth transitions...</p>
</div>

CSS

.transform-demo {
  width: 100px;
  height: 100px;
  background: var(--brand);
  transition: transform 0.3s ease;
  cursor: pointer;
}

.transform-demo:hover {
  transform: scale(1.2) rotate(10deg);
}

.transition-demo {
  background: var(--surface);
  padding: var(--space-4);
  border-radius: var(--radius-1);
  transition: all 0.3s ease;
}

.transition-demo:hover {
  background: var(--brand-light);
  transform: translateY(-2px);
  box-shadow: var(--shadow-md);
}

.scale-demo:hover { transform: scale(1.2); }
.rotate-demo:hover { transform: rotate(45deg); }
.translate-demo:hover { transform: translate(10px, -10px); }
.skew-demo:hover { transform: skew(10deg, 5deg); }

Notes

  • Transitions: Smooth changes between property values
  • Transform: Move, rotate, scale, or skew elements
  • Duration: How long the transition takes
  • Timing function: How the transition progresses (ease, linear, etc.)
  • Delay: Wait time before transition starts
  • Transform-origin: Point around which transforms occur
  • Performance: Transforms are hardware-accelerated
  • Accessibility: Respect prefers-reduced-motion
