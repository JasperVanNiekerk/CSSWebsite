Google Fonts

Learn how to integrate Google Fonts into your projects. Google Fonts provides a vast collection of free, open-source fonts that are optimized for web use. They're easy to implement and provide excellent performance and accessibility.

Live Demo

Google Fonts in Action

This page uses two Google Fonts: Inter (sans-serif) and Playfair Display (serif). Notice how they enhance the typography:

Inter Font (Sans-serif)

Inter is a modern, highly readable sans-serif font designed specifically for computer screens. It's perfect for body text, interfaces, and digital content.

Playfair Display Font (Serif)

Playfair Display is an elegant serif font with high contrast and distinctive character. It's ideal for headings, titles, and decorative text.

Google Fonts Implementation

This page uses the following Google Fonts implementation:

  • Preconnect: <link rel="preconnect" href="https://fonts.googleapis.com">
  • Font Link: <link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&family=Playfair+Display:wght@400;500;600;700&display=swap" rel="stylesheet">
  • CSS Usage: font-family: 'Inter', sans-serif;

Code

HTML

<!-- Preconnect to Google Fonts for better performance -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>

<!-- Load Google Fonts -->
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&family=Playfair+Display:wght@400;500;600;700&display=swap" rel="stylesheet">

<div class="font-demo google-font-inter">
  <h4>Inter Font (Sans-serif)</h4>
  <p>Inter is a modern, highly readable sans-serif font...</p>
</div>

<div class="font-demo google-font-playfair">
  <h4>Playfair Display Font (Serif)</h4>
  <p>Playfair Display is an elegant serif font...</p>
</div>

CSS

/* Apply Google Fonts with fallbacks */
.google-font-inter {
  font-family: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
}

.google-font-playfair {
  font-family: 'Playfair Display', Georgia, 'Times New Roman', serif;
}

/* System font stack for comparison */
.system-font {
  font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
}

/* Font weight variants */
.weight-300 { font-weight: 300; }
.weight-400 { font-weight: 400; }
.weight-500 { font-weight: 500; }
.weight-600 { font-weight: 600; }
.weight-700 { font-weight: 700; }

/* Font style variants */
.style-normal { font-style: normal; }
.style-italic { font-style: italic; }

/* Font demo styling */
.font-demo {
  font-size: var(--fs-lg);
  line-height: 1.6;
  margin: var(--space-4) 0;
  padding: var(--space-4);
  background: var(--font-demo-bg);
  border-radius: var(--radius-1);
  border-left: 4px solid var(--brand);
}

.font-demo h4 {
  color: var(--brand);
  margin-bottom: var(--space-2);
  font-size: var(--fs-sm);
  text-transform: uppercase;
  letter-spacing: 0.05em;
}

Notes

  • Preconnect: Use rel="preconnect" to improve loading performance
  • Display swap: Use display=swap to show fallback fonts while loading
  • Fallback fonts: Always provide system font fallbacks
  • Font weights: Only load the weights you actually use
  • Performance: Google Fonts are optimized and cached globally
  • Accessibility: Google Fonts meet web accessibility standards
  • Self-hosting: Consider self-hosting for better privacy and control
  • Font pairing: Choose fonts that work well together
  • Loading strategy: Use font-display: swap for better UX
