Column Variations

Different column configurations and styling options:

Two Columns

This content is displayed in two columns with a moderate gap between them. It's perfect for shorter articles and content that doesn't need as much horizontal space.

Three Columns

Three columns provide a good balance between readability and space efficiency. This is often used for newspaper-style layouts and long-form content.

Four Columns

Four columns are ideal for wide screens and content that needs to be very compact. This works well for reference materials and technical documentation.

Auto Columns

Auto columns adjust the number of columns based on the available width. This content will automatically flow into as many columns as fit the container width.

Multi-column Properties

  • column-count: number | auto - Number of columns
  • column-width: length | auto - Width of each column
  • column-gap: length - Space between columns
  • column-rule: width style color - Line between columns
  • column-span: none | all - Span across columns
  • break-before: auto | column | page - Break before element
  • break-after: auto | column | page - Break after element
  • break-inside: auto | avoid - Break inside element

CSS

/* Basic multi-column layout */
.column-demo {
  column-count: 3;
  column-gap: var(--space-6);
  column-rule: 1px solid var(--border);
  margin: var(--space-4) 0;
  padding: var(--space-4);
  background: var(--column-demo-bg);
  border-radius: var(--radius-1);
}

.column-demo h3 {
  column-span: all;
  margin-bottom: var(--space-4);
  color: var(--brand);
  text-align: center;
}

/* Column variations */
.column-2 { 
  column-count: 2; 
  column-gap: var(--space-4); 
}

.column-3 { 
  column-count: 3; 
  column-gap: var(--space-4); 
}

.column-4 { 
  column-count: 4; 
  column-gap: var(--space-3); 
}

.column-auto { 
  column-count: auto; 
  column-width: 200px; 
  column-gap: var(--space-4); 
}

/* Column styling */
.column-rules {
  column-rule: 2px solid var(--brand);
  column-rule-style: dashed;
}

.column-gaps {
  column-gap: var(--space-8);
}

.column-spans {
  column-span: all;
  background: var(--brand-light);
  padding: var(--space-2);
  border-radius: var(--radius-1);
  margin: var(--space-2) 0;
  text-align: center;
  font-weight: bold;
  color: var(--brand);
}

/* Column breaks */
.break-inside-avoid { break-inside: avoid; }
.break-before-column { break-before: column; }
.break-after-column { break-after: column; }

/* Responsive columns */
.responsive-columns {
  column-count: 1;
  column-gap: var(--space-4);
}

@media (min-width: 768px) {
  .responsive-columns {
    column-count: 2;
  }
}

@media (min-width: 1024px) {
  .responsive-columns {
    column-count: 3;
  }
}

Notes

  • Column-count: Sets the number of columns (or use auto)
  • Column-width: Sets the width of each column (or use auto)
  • Column-gap: Controls spacing between columns
  • Column-rule: Adds lines between columns
  • Column-span: Allows elements to span across columns
  • Break properties: Control where content breaks between columns
  • Responsive design: Adjust column count for different screen sizes
  • Content flow: Content automatically flows from one column to the next
  • Browser support: Good support in modern browsers
