Code

HTML

<div class="border-radius-demo prefixed">Prefixed</div>
<div class="border-radius-demo modern">Modern</div>

CSS

/* With vendor prefixes */
.border-radius-demo.prefixed {
  -webkit-border-radius: 8px;
  -moz-border-radius: 8px;
  border-radius: 8px;
}

/* Modern CSS (no prefixes needed) */
.border-radius-demo.modern {
  border-radius: 8px;
}

/* CSS Reset example */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

Notes

  • Vendor prefixes: -webkit-, -moz-, -ms-, -o- for browser compatibility
  • Order matters: Prefixes first, standard property last
  • Modern browsers: Most prefixes are no longer needed
  • CSS Reset: Normalizes browser defaults for consistency
  • Box-sizing: Include border-box for predictable sizing
  • Use tools: Autoprefixer can add prefixes automatically
  • Test thoroughly: Always test in target browsers
