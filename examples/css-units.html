CSS Units

Learn about different CSS units for sizing elements. Understanding when to use each unit type is crucial for creating responsive, accessible, and maintainable designs. Each unit has specific use cases and behaviors.

Live Demo

Absolute Units

Fixed-size units that don't change based on other properties:

Pixels (px)

100px × 60px

Fixed size, not responsive

Relative Units

Units that are calculated relative to other values:

Percentage (%)

50% width

Relative to parent element

Em (em)

10em × 3em

Relative to parent font-size

Rem (rem)

10rem × 3rem

Relative to root font-size

Viewport Units

Units relative to the viewport size:

Viewport Width (vw)

20vw width

1vw = 1% of viewport width

Viewport Height (vh)

20vw × 10vh

1vh = 1% of viewport height

Viewport Min (vmin)

20vmin × 20vmin

1vmin = 1% of smaller viewport dimension

Viewport Max (vmax)

20vmax × 20vmax

1vmax = 1% of larger viewport dimension

Unit Comparison

Same content with different units - resize your browser to see the difference:

Fixed: 300px width
Responsive: 50% width
Scalable: 20rem width
Viewport: 50vw width

Unit Characteristics

  • px: Fixed size - Always the same size regardless of context
  • %: Parent relative - Percentage of parent element's size
  • em: Font relative - Multiplier of parent element's font-size
  • rem: Root relative - Multiplier of root element's font-size
  • vw/vh: Viewport relative - Percentage of viewport width/height
  • vmin/vmax: Viewport relative - Percentage of smaller/larger viewport dimension

Code

HTML

<div class="unit-card">
  <h4>Pixels (px)</h4>
  <div class="demo-box px-demo">100px × 60px</div>
  <p>Fixed size, not responsive</p>
</div>

<div class="unit-card">
  <h4>Percentage (%)</h4>
  <div class="demo-box percent-demo">50% width</div>
  <p>Relative to parent element</p>
</div>

<div class="unit-card">
  <h4>Em (em)</h4>
  <div class="demo-box em-demo">10em × 3em</div>
  <p>Relative to parent font-size</p>
</div>

<div class="unit-card">
  <h4>Rem (rem)</h4>
  <div class="demo-box rem-demo">10rem × 3rem</div>
  <p>Relative to root font-size</p>
</div>

CSS

.unit-card {
  background: var(--unit-demo-bg);
  border: 1px solid var(--border);
  border-radius: var(--radius-1);
  padding: var(--space-4);
  margin: var(--space-2);
  text-align: center;
  font-size: var(--fs-sm);
  display: inline-block;
  vertical-align: top;
}

/* Pixel units - fixed size */
.px-demo { 
  width: 100px; 
  height: 60px; 
}

/* Percentage units - relative to parent */
.percent-demo { 
  width: 50%; 
  height: 60px; 
}

/* Em units - relative to parent font-size */
.em-demo { 
  width: 10em; 
  height: 3em; 
  font-size: 1em; 
}

/* Rem units - relative to root font-size */
.rem-demo { 
  width: 10rem; 
  height: 3rem; 
  font-size: 1rem; 
}

/* Viewport width units */
.vw-demo { 
  width: 20vw; 
  height: 60px; 
}

/* Viewport height units */
.vh-demo { 
  width: 20vw; 
  height: 10vh; 
}

/* Viewport minimum units */
.vmin-demo { 
  width: 20vmin; 
  height: 20vmin; 
}

/* Viewport maximum units */
.vmax-demo { 
  width: 20vmax; 
  height: 20vmax; 
}

/* Responsive comparison */
.responsive-demo .px-box { width: 300px; }
.responsive-demo .percent-box { width: 50%; }
.responsive-demo .rem-box { width: 20rem; }
.responsive-demo .vw-box { width: 50vw; }

Notes

  • Use px for: Borders, small fixed elements, precise positioning
  • Use % for: Layout widths, responsive containers
  • Use em for: Typography that scales with parent font-size
  • Use rem for: Consistent typography scale across the site
  • Use vw/vh for: Full-screen layouts, responsive typography
  • Use vmin/vmax for: Square elements that scale with viewport
  • Accessibility: rem units respect user's font-size preferences
  • Performance: Relative units can cause layout recalculations
  • Browser support: All modern browsers support these units
