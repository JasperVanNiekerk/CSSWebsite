Z-index & Opacity

Learn how to control the stacking order of elements with z-index and create transparency effects with opacity. These properties are essential for creating layered interfaces and visual effects.

Live Demo

Z-index Layering

These boxes demonstrate z-index stacking. Higher z-index values appear on top:

z: 1
z: 2
z: 3

Z-index Values:

  • Red box: z-index: 1 (bottom layer)
  • Teal box: z-index: 2 (middle layer)
  • Blue box: z-index: 3 (top layer)

Note: Z-index only works on positioned elements (relative, absolute, fixed, sticky).

Opacity Transparency

Hover over these boxes to see opacity changes. Opacity values range from 0 (transparent) to 1 (opaque):

0%
25%
50%
75%
100%

Opacity Values:

  • 0 (0%): Completely transparent (invisible)
  • 0.25 (25%): Very transparent
  • 0.5 (50%): Semi-transparent
  • 0.75 (75%): Mostly opaque
  • 1 (100%): Completely opaque (default)

Code

HTML

<div class="z-index-demo">
  <div class="z-index-box box-1">z: 1</div>
  <div class="z-index-box box-2">z: 2</div>
  <div class="z-index-box box-3">z: 3</div>
</div>

<div class="opacity-demo">
  <div class="opacity-box opacity-0">0%</div>
  <div class="opacity-box opacity-25">25%</div>
  <div class="opacity-box opacity-50">50%</div>
  <div class="opacity-box opacity-75">75%</div>
  <div class="opacity-box opacity-100">100%</div>
</div>

CSS

.z-index-demo {
  position: relative;
  width: 300px;
  height: 200px;
  margin: var(--space-4) auto;
  background: var(--surface);
  border: 2px solid var(--border);
  border-radius: var(--radius-1);
}

.z-index-box {
  position: absolute;
  width: 80px;
  height: 80px;
  display: flex;
  align-items: center;
  justify-content: center;
  color: white;
  font-weight: bold;
  border-radius: var(--radius-1);
  transition: all 0.3s ease;
  cursor: pointer;
}

.box-1 {
  background: var(--box-1);
  top: 20px;
  left: 20px;
  z-index: 1;
}

.box-2 {
  background: var(--box-2);
  top: 40px;
  left: 40px;
  z-index: 2;
}

.box-3 {
  background: var(--box-3);
  top: 60px;
  left: 60px;
  z-index: 3;
}

.opacity-box {
  width: 100px;
  height: 60px;
  background: var(--brand);
  color: white;
  display: inline-flex;
  align-items: center;
  justify-content: center;
  margin: var(--space-2);
  border-radius: var(--radius-1);
  font-weight: bold;
  transition: opacity 0.3s ease;
  cursor: pointer;
}

.opacity-box:hover {
  opacity: 0.5;
}

.opacity-0 { opacity: 0; }
.opacity-25 { opacity: 0.25; }
.opacity-50 { opacity: 0.5; }
.opacity-75 { opacity: 0.75; }
.opacity-100 { opacity: 1; }

Notes

  • Z-index: Controls stacking order of positioned elements
  • Higher z-index values appear on top of lower values
  • Z-index only works on positioned elements (relative, absolute, fixed, sticky)
  • Opacity: Controls transparency from 0 (transparent) to 1 (opaque)
  • Opacity affects the entire element and all its children
  • Use rgba() or hsla() for background color transparency instead of opacity when possible
  • Z-index stacking context: Elements with z-index create new stacking contexts
  • Negative z-index values can place elements behind their parent's background
