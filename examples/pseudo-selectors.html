Pseudo Selectors

Learn about CSS pseudo-selectors that target elements based on their position, state, or relationship to other elements. These selectors provide powerful ways to style elements without adding extra classes or IDs.

Live Demo

Pseudo Selector Examples

Hover over these list items to see different pseudo-selectors in action:

  • :first-child - First item in the list
  • :nth-child(odd) - Every odd item
  • :hover - Hover over me!
  • :nth-child(3n) - Every third item
  • :not(:first-child):not(:last-child) - Middle items
  • :last-child - Last item in the list

Code

HTML

<div class="pseudo-demo">
  <ul>
    <li>:first-child - First item in the list</li>
    <li>:nth-child(odd) - Every odd item</li>
    <li>:hover - Hover over me!</li>
    <li>:nth-child(3n) - Every third item</li>
    <li>:not(:first-child):not(:last-child) - Middle items</li>
    <li>:last-child - Last item in the list</li>
  </ul>
</div>

CSS

.pseudo-demo li:first-child {
  background: var(--brand-light);
  color: var(--brand);
  font-weight: bold;
}

.pseudo-demo li:nth-child(odd) {
  border-left: 3px solid var(--brand);
}

.pseudo-demo li:hover {
  background: var(--brand);
  color: white;
  transform: translateX(4px);
}

.pseudo-demo li:last-child {
  border-bottom: 2px dashed var(--border);
}

.pseudo-demo li:nth-child(3n) {
  font-style: italic;
}

.pseudo-demo li:not(:first-child):not(:last-child) {
  opacity: 0.9;
}

Notes

  • :first-child - Targets the first child element
  • :last-child - Targets the last child element
  • :nth-child(n) - Targets elements at position n
  • :nth-child(odd) - Targets odd-positioned elements
  • :nth-child(even) - Targets even-positioned elements
  • :hover - Targets elements being hovered
  • :not() - Targets elements that don't match the selector
  • Cascade matters - Later rules override earlier ones
