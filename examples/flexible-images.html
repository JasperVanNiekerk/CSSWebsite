Flexible Images

Learn how to make images responsive and prevent layout breaking on small screens. Flexible images are essential for responsive web design and provide a better user experience across all devices.

Image Sizing Techniques

Different approaches to making images flexible:

Basic Responsive Image

max-width: 100%; height: auto;

This is the most common approach for responsive images.

max-width: 100%

Aspect Ratio Container

aspect-ratio: 16 / 9;

Maintains consistent aspect ratio regardless of content.

16:9 Aspect Ratio

Object Fit

object-fit: cover;

Fills the container while maintaining aspect ratio.

object-fit: cover

Flexible Image Best Practices

  • Always use max-width: 100% to prevent images from exceeding container width
  • Include height: auto to maintain aspect ratio
  • Use object-fit for images that need to fill specific dimensions
  • Consider aspect-ratio for consistent image containers
  • Test on mobile devices to ensure images display properly
  • Use srcset for different image sizes on different devices
  • Include alt text for accessibility
  • Optimize image file sizes for better performance

Code

HTML

<!-- Responsive image -->
<img src="image.jpg" alt="Description" class="responsive-image">

<!-- Fixed width image (avoid) -->
<img src="image.jpg" alt="Description" class="fixed-image">

<!-- Aspect ratio container -->
<div class="image-aspect-ratio">
  <img src="image.jpg" alt="Description">
</div>

<!-- Object fit image -->
<img src="image.jpg" alt="Description" class="image-object-fit">

CSS

/* Responsive image */
.responsive-image {
  max-width: 100%;
  height: auto;
  border-radius: var(--radius-1);
  box-shadow: var(--shadow-sm);
}

/* Fixed width image (avoid) */
.fixed-image {
  width: 800px;
  height: 400px;
  border-radius: var(--radius-1);
  box-shadow: var(--shadow-sm);
}

/* Aspect ratio container */
.image-aspect-ratio {
  aspect-ratio: 16 / 9;
  background: var(--brand-light);
  border-radius: var(--radius-1);
  display: flex;
  align-items: center;
  justify-content: center;
  color: var(--brand);
  font-weight: bold;
  margin: var(--space-2) 0;
}

/* Object fit image */
.image-object-fit {
  width: 100%;
  height: 200px;
  object-fit: cover;
  border-radius: var(--radius-1);
  box-shadow: var(--shadow-sm);
}

/* Image container */
.image-container {
  background: var(--surface);
  padding: var(--space-4);
  border-radius: var(--radius-1);
  margin: var(--space-2) 0;
  border: 1px solid var(--border);
  overflow-x: auto;
}

Notes

  • max-width: 100% - Prevents images from exceeding container width
  • height: auto - Maintains aspect ratio when width changes
  • object-fit: cover - Fills container while maintaining aspect ratio
  • aspect-ratio - Creates consistent image containers
  • srcset attribute - Provides different image sizes for different devices
  • alt text - Essential for accessibility and SEO
  • File optimization - Compress images for better performance
  • Test on devices - Always test on actual mobile devices
