Responsive Layout

Learn how to create complete responsive layouts that adapt to different screen sizes. This example shows a sidebar and main content area that stacks on mobile devices.

Live Demo

Responsive Sidebar Layout

This layout demonstrates a sidebar and main content area that adapts to different screen sizes:

Sidebar

This sidebar contains navigation and secondary content. On mobile devices, it will stack above the main content.

  • Navigation Item 1
  • Navigation Item 2
  • Navigation Item 3

Main Content

This is the main content area. It takes up the remaining space and adapts to different screen sizes. On desktop, it sits next to the sidebar. On mobile, it stacks below.

Resize your browser window to see how the layout changes from desktop to mobile view.

Responsive Layout Techniques

  • Flexbox: Use display: flex for flexible layouts
  • Flex properties: flex: 1 for growing content, flex: 0 0 200px for fixed sidebar
  • Media queries: Change layout direction with flex-direction: column
  • Gap property: Use gap for consistent spacing
  • Min-height: Ensure layout has minimum height
  • Test breakpoints: Check layout at different screen sizes

Code

HTML

<div class="layout-demo">
  <div class="sidebar">
    <h4>Sidebar</h4>
    <p>This sidebar contains navigation...</p>
    <ul>
      <li>Navigation Item 1</li>
      <li>Navigation Item 2</li>
      <li>Navigation Item 3</li>
    </ul>
  </div>
  
  <div class="content">
    <h4>Main Content</h4>
    <p>This is the main content area...</p>
  </div>
</div>

CSS

.layout-demo {
  display: flex;
  gap: var(--space-4);
  margin: var(--space-4) 0;
  min-height: 200px;
}

.layout-demo .sidebar {
  flex: 0 0 200px;
  background: var(--surface);
  padding: var(--space-4);
  border-radius: var(--radius-1);
  border: 1px solid var(--border);
}

.layout-demo .content {
  flex: 1;
  background: var(--bg);
  padding: var(--space-4);
  border: 1px solid var(--border);
  border-radius: var(--radius-1);
}

@media (max-width: 767px) {
  .layout-demo {
    flex-direction: column;
  }
  
  .layout-demo .sidebar {
    flex: none;
  }
}

Notes

  • Flexbox layout: Use flexbox for flexible, responsive layouts
  • Sidebar + content: Common pattern for web layouts
  • Mobile stacking: Stack elements vertically on small screens
  • Flex properties: Control how elements grow and shrink
  • Gap spacing: Use gap property for consistent spacing
  • Test breakpoints: Check layout at different screen sizes
  • Accessibility: Ensure layout works with screen readers
