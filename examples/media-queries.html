Media Queries

Learn how to apply different styles based on screen size, device characteristics, and other media features. Media queries are the foundation of responsive web design, allowing you to create layouts that adapt to different devices and screen sizes.

HTML

<div class="responsive-demo">
  <p>Resize your browser window to see how this box responds to different screen sizes!</p>
</div>

<div class="breakpoint-demo">
  <div class="box">
    <h4>Box 1</h4>
    <p>Mobile: Full width</p>
  </div>
  <div class="box">
    <h4>Box 2</h4>
    <p>Tablet: 2 columns</p>
  </div>
  <div class="box">
    <h4>Box 3</h4>
    <p>Desktop: 3 columns</p>
  </div>
</div>

<div class="orientation-demo">
  <p>Rotate your device or resize your browser window to see orientation changes!</p>
</div>

CSS

/* Mobile first approach - base styles */
.responsive-demo {
  background: var(--mobile-bg);
  border: 2px solid var(--brand);
  padding: var(--space-4);
  margin: var(--space-4) 0;
  text-align: center;
  font-weight: bold;
  color: var(--brand);
  border-radius: var(--radius-1);
  transition: all 0.3s ease;
}

.responsive-demo::before {
  content: "Mobile (100% width)";
}

/* Tablet styles - 768px and up */
@media (min-width: 768px) {
  .responsive-demo {
    background: var(--tablet-bg);
    width: 50%;
    margin: var(--space-4) auto;
  }
  
  .responsive-demo::before {
    content: "Tablet (50% width)";
  }
}

/* Desktop styles - 1024px and up */
@media (min-width: 1024px) {
  .responsive-demo {
    background: var(--desktop-bg);
    width: 33.333%;
  }
  
  .responsive-demo::before {
    content: "Desktop (33% width)";
  }
}

/* Responsive grid - mobile first */
.breakpoint-demo {
  display: grid;
  gap: var(--space-4);
  margin: var(--space-4) 0;
  grid-template-columns: 1fr; /* Mobile: 1 column */
}

/* Tablet: 2 columns */
@media (min-width: 768px) {
  .breakpoint-demo {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* Desktop: 3 columns */
@media (min-width: 1024px) {
  .breakpoint-demo {
    grid-template-columns: repeat(3, 1fr);
  }
}

/* Orientation detection */
.orientation-demo::before {
  content: "Portrait orientation";
  color: var(--brand);
  font-weight: bold;
}

@media (orientation: landscape) {
  .orientation-demo::before {
    content: "Landscape orientation";
  }
}

Notes

  • Mobile-first approach: Start with mobile styles, then enhance for larger screens
  • Common breakpoints: 768px (tablet), 1024px (desktop), 1200px (large desktop)
  • min-width: Styles apply when screen is at least this wide
  • max-width: Styles apply when screen is at most this wide
  • orientation: portrait or landscape detection
  • Always include viewport meta tag: <meta name="viewport" content="width=device-width, initial-scale=1">
  • Use relative units (%, em, rem) instead of fixed pixels for better responsiveness
  • Test on actual devices, not just browser resizing
