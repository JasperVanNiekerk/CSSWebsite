CSS Variables

Learn how to use CSS custom properties (variables) to create maintainable, dynamic styles. Variables allow you to store values and reuse them throughout your CSS, making it easier to maintain consistent designs.

Code

HTML

<div class="variable-demo">
  <h3>Variable-powered Heading</h3>
  <p>This heading uses CSS variables...</p>
  <button>Variable Button</button>
  <div class="box"></div>
</div>

CSS

.variable-demo {
  --demo-color: #ff6b6b;
  --demo-size: 1.5rem;
  --demo-spacing: 2rem;
  padding: var(--space-4);
  background: var(--surface);
  border-radius: var(--radius-1);
  margin: var(--space-4) 0;
}

.variable-demo h3 {
  color: var(--demo-color);
  font-size: var(--demo-size);
}

.variable-demo button {
  background: var(--demo-color);
  color: white;
  border: none;
  padding: var(--space-2) var(--space-4);
  border-radius: var(--radius-1);
  cursor: pointer;
  margin: var(--space-2);
}

.variable-demo .box {
  width: var(--demo-spacing);
  height: var(--demo-spacing);
  background: var(--demo-color);
  margin: var(--space-2);
  border-radius: var(--radius-1);
}

Notes

  • Define with --: --variable-name: value;
  • Use with var(): property: var(--variable-name);
  • Scope: Variables inherit from parent elements
  • :root scope: Global variables available everywhere
  • Fallback values: var(--var, fallback)
  • Dynamic updates: Variables can be changed with JavaScript
  • Browser support: Excellent support in modern browsers
  • Performance: Variables are computed at runtime
