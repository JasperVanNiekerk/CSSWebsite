Borders

Border Shorthand vs Individual Properties

Two ways to achieve the same result:

border: 3px solid #0066cc;
Individual properties

Shorthand: border: width style color;

Individual: border-width, border-style, border-color

CSS

.border-demo {
  width: 100px;
  height: 60px;
  margin: var(--space-2);
  display: inline-flex;
  align-items: center;
  justify-content: center;
  font-size: var(--fs-sm);
  text-align: center;
  background: var(--border-demo-bg);
}

/* Border styles */
.border-demo.solid { border: 2px solid var(--brand); }
.border-demo.dashed { border: 2px dashed var(--brand); }
.border-demo.dotted { border: 2px dotted var(--brand); }
.border-demo.double { border: 4px double var(--brand); }
.border-demo.groove { border: 4px groove var(--brand); }
.border-demo.ridge { border: 4px ridge var(--brand); }
.border-demo.inset { border: 4px inset var(--brand); }
.border-demo.outset { border: 4px outset var(--brand); }
.border-demo.none { border: none; }
.border-demo.hidden { border: 2px hidden var(--brand); }

/* Border widths */
.border-width-demo .thin { border-width: 1px; }
.border-width-demo .medium { border-width: 3px; }
.border-width-demo .thick { border-width: 6px; }
.border-width-demo .custom { border-width: 2px 4px 6px 8px; }

/* Border colors */
.border-color-demo .red { border-color: #ff6b6b; }
.border-color-demo .green { border-color: #4ecdc4; }
.border-color-demo .blue { border-color: #45b7d1; }
.border-color-demo .multi { border-color: #ff6b6b #4ecdc4 #45b7d1 #f9ca24; }

/* Individual sides */
.border-side-demo .top { border-top: 3px solid var(--brand); }
.border-side-demo .right { border-right: 3px solid var(--brand); }
.border-side-demo .bottom { border-bottom: 3px solid var(--brand); }
.border-side-demo .left { border-left: 3px solid var(--brand); }

/* Border radius */
.border-radius-demo .small { border-radius: 4px; }
.border-radius-demo .medium { border-radius: 8px; }
.border-radius-demo .large { border-radius: 16px; }
.border-radius-demo .circle { border-radius: 50%; }
.border-radius-demo .custom { border-radius: 8px 16px 4px 20px; }

/* Shorthand vs individual */
.border-shorthand-demo .shorthand { border: 3px solid var(--brand); }
.border-shorthand-demo .individual {
  border-width: 3px;
  border-style: solid;
  border-color: var(--brand);
}

Notes

  • Border shorthand: border: width style color; (order matters)
  • Individual properties: border-width, border-style, border-color
  • Side-specific: border-top, border-right, border-bottom, border-left
  • Border-radius: Creates rounded corners, can be different for each corner
  • Border-box: Borders are included in element's total width/height by default
  • Transparent borders: Use border-color: transparent for invisible borders
  • Border-collapse: Use border-collapse: collapse for table borders
  • Box-shadow: Alternative to borders for some visual effects
