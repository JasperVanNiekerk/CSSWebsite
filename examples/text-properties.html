Text Properties

Learn how to control typography and text appearance with CSS. These properties allow you to create readable, visually appealing text that enhances user experience and communicates your message effectively.

White Space

white-space

Normal: This text has multiple spaces and line breaks that are collapsed.
Nowrap: This text will not wrap to the next line even if it's very long and exceeds the container width.
Pre: This text preserves spaces and line breaks exactly as written.
Pre-wrap: This text preserves spaces and line breaks but will wrap when needed.
Pre-line: This text preserves line breaks but collapses spaces and wraps when needed.

CSS

.text-demo {
  margin: var(--space-4) 0;
  padding: var(--space-3);
  border-left: 3px solid var(--brand);
  background: var(--text-demo-bg);
  border-radius: var(--radius-1);
}

/* Font sizes */
.font-size-demo .small { font-size: 12px; }
.font-size-demo .medium { font-size: 16px; }
.font-size-demo .large { font-size: 24px; }
.font-size-demo .xlarge { font-size: 32px; }

/* Font styles */
.font-style-demo .normal { font-style: normal; }
.font-style-demo .italic { font-style: italic; }
.font-style-demo .oblique { font-style: oblique; }

/* Font weights */
.font-weight-demo .light { font-weight: 300; }
.font-weight-demo .normal { font-weight: 400; }
.font-weight-demo .medium { font-weight: 500; }
.font-weight-demo .bold { font-weight: 700; }
.font-weight-demo .black { font-weight: 900; }

/* Text alignment */
.text-align-demo .left { text-align: left; }
.text-align-demo .center { text-align: center; }
.text-align-demo .right { text-align: right; }
.text-align-demo .justify { text-align: justify; }

/* Text decoration */
.text-decoration-demo .none { text-decoration: none; }
.text-decoration-demo .underline { text-decoration: underline; }
.text-decoration-demo .overline { text-decoration: overline; }
.text-decoration-demo .line-through { text-decoration: line-through; }
.text-decoration-demo .underline-overline { text-decoration: underline overline; }

/* Letter spacing */
.letter-spacing-demo .tight { letter-spacing: -0.5px; }
.letter-spacing-demo .normal { letter-spacing: normal; }
.letter-spacing-demo .wide { letter-spacing: 2px; }
.letter-spacing-demo .wider { letter-spacing: 4px; }

/* Text transform */
.text-transform-demo .none { text-transform: none; }
.text-transform-demo .uppercase { text-transform: uppercase; }
.text-transform-demo .lowercase { text-transform: lowercase; }
.text-transform-demo .capitalize { text-transform: capitalize; }

/* White space */
.white-space-demo .normal { white-space: normal; }
.white-space-demo .nowrap { white-space: nowrap; }
.white-space-demo .pre { white-space: pre; }
.white-space-demo .pre-wrap { white-space: pre-wrap; }
.white-space-demo .pre-line { white-space: pre-line; }

/* Vertical align */
.vertical-align-demo .baseline { vertical-align: baseline; }
.vertical-align-demo .top { vertical-align: top; }
.vertical-align-demo .middle { vertical-align: middle; }
.vertical-align-demo .bottom { vertical-align: bottom; }
.vertical-align-demo .sub { vertical-align: sub; }
.vertical-align-demo .super { vertical-align: super; }

Notes

  • Font-size: Use relative units (em, rem) for better scalability
  • Font-weight: Values range from 100-900, but not all weights are available for every font
  • Text-align: Only works on block-level elements
  • Text-decoration: Can be combined (underline overline)
  • Letter-spacing: Can use negative values for tighter spacing
  • Text-transform: Useful for consistent casing without changing HTML
  • White-space: pre preserves all whitespace, pre-wrap adds wrapping
  • Vertical-align: Only works on inline and table-cell elements
  • Line-height: Controls spacing between lines of text
