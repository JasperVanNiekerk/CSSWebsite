Mobile-first

Learn how to build responsive designs starting from mobile and enhancing upward. Mobile-first design ensures your site works well on all devices and provides better performance.

Live Demo

Mobile-first Layout

This layout starts with mobile styles and enhances for larger screens:

Item 1

Mobile: Full width

Item 2

Tablet: 2 columns

Item 3

Desktop: 3 columns

Mobile-first Benefits

  • Better performance - Smaller initial CSS bundle
  • Progressive enhancement - Start simple, add complexity
  • Touch-friendly - Designed for mobile interactions
  • Faster loading - Optimized for mobile networks
  • Better UX - Works well on all devices

Code

HTML

<div class="mobile-layout">
  <div class="item">
    <h5>Item 1</h5>
    <p>Mobile: Full width</p>
  </div>
  <div class="item">
    <h5>Item 2</h5>
    <p>Tablet: 2 columns</p>
  </div>
  <div class="item">
    <h5>Item 3</h5>
    <p>Desktop: 3 columns</p>
  </div>
</div>

CSS

/* Mobile-first base styles */
.mobile-layout {
  display: grid;
  gap: var(--space-4);
  margin: var(--space-4) 0;
  grid-template-columns: 1fr; /* Mobile: 1 column */
}

/* Tablet styles */
@media (min-width: 768px) {
  .mobile-layout {
    grid-template-columns: 1fr 1fr; /* Tablet: 2 columns */
  }
}

/* Desktop styles */
@media (min-width: 1024px) {
  .mobile-layout {
    grid-template-columns: 1fr 1fr 1fr; /* Desktop: 3 columns */
  }
}

Notes

  • Start mobile: Design for smallest screen first
  • Progressive enhancement: Add features for larger screens
  • min-width media queries: Use min-width, not max-width
  • Touch targets: Make buttons and links touch-friendly
  • Performance: Optimize for mobile networks
  • Content priority: Show most important content first
  • Test on devices: Always test on actual mobile devices
