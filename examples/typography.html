Typography Scale

Learn how to create a consistent, harmonious typography scale using relative units. A well-designed typography scale ensures readability, visual hierarchy, and professional appearance across your entire website.

Line Height Variations

Different line heights for different content types:

Tight Line Height (1.2)

This text has tight line spacing, which is often used for headings and short text blocks. It creates a more compact appearance but can be harder to read for longer content.

Normal Line Height (1.5)

This text has normal line spacing, which is ideal for body text and most content. It provides good readability while maintaining a clean, professional appearance.

Loose Line Height (1.8)

This text has loose line spacing, which is often used for accessibility or when you want a more relaxed, spacious feel. It's great for long-form content and improves readability.

Typography Scale Best Practices

  • Use relative units: rem for font sizes, em for line heights
  • Establish hierarchy: Clear size differences between heading levels
  • Maintain proportions: Use a consistent ratio (1.25, 1.5, or 1.618)
  • Consider line height: 1.4-1.6 for body text, 1.2-1.4 for headings
  • Test readability: Ensure text is comfortable to read at all sizes
  • Use CSS custom properties: Define your scale in variables for consistency
  • Responsive scaling: Adjust scale for different screen sizes

CSS

/* Typography scale using relative units */
.typography-scale h1 {
  font-size: 2.5rem;
  line-height: 1.2;
  margin-bottom: var(--space-4);
  color: var(--brand);
}

.typography-scale h2 {
  font-size: 2rem;
  line-height: 1.3;
  margin-bottom: var(--space-3);
  color: var(--text);
}

.typography-scale h3 {
  font-size: 1.5rem;
  line-height: 1.4;
  margin-bottom: var(--space-3);
  color: var(--text);
}

.typography-scale h4 {
  font-size: 1.25rem;
  line-height: 1.4;
  margin-bottom: var(--space-2);
  color: var(--text);
}

.typography-scale h5 {
  font-size: 1.125rem;
  line-height: 1.5;
  margin-bottom: var(--space-2);
  color: var(--text);
}

.typography-scale h6 {
  font-size: 1rem;
  line-height: 1.5;
  margin-bottom: var(--space-2);
  color: var(--muted);
}

.typography-scale p {
  font-size: 1rem;
  line-height: 1.6;
  margin-bottom: var(--space-4);
  color: var(--text);
}

.typography-scale .small {
  font-size: 0.875rem;
  line-height: 1.5;
  color: var(--muted);
}

.typography-scale .large {
  font-size: 1.125rem;
  line-height: 1.6;
}

/* Line height variations */
.line-height-tight { line-height: 1.2; }
.line-height-normal { line-height: 1.5; }
.line-height-loose { line-height: 1.8; }

/* Typography demo styling */

  padding: var(--space-6);

  margin: var(--space-4) 0;

}

Notes

  • Relative units: Use rem for font sizes, em for line heights
  • Consistent ratio: Use a mathematical ratio (1.25, 1.5, or 1.618) for scale
  • Line height: 1.4-1.6 for body text, 1.2-1.4 for headings
  • CSS custom properties: Define your scale in variables for consistency
  • Responsive design: Adjust scale for different screen sizes
  • Accessibility: Ensure sufficient contrast and readable sizes
  • Testing: Test your scale with real content, not just headings
  • Modular scale: Consider using a modular scale for mathematical consistency
  • Vertical rhythm: Use consistent spacing between elements
