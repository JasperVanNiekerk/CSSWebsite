Web Fonts (@font-face)

Learn how to embed custom fonts using @font-face declarations. This allows you to use fonts that aren't installed on users' systems, giving you complete control over typography while maintaining good performance and accessibility.

@font-face Best Practices

  • Use WOFF2 format: Best compression and browser support
  • Include fallbacks: Always provide system font fallbacks
  • Use font-display: swap: Improves loading performance
  • Preload critical fonts: Use <link rel="preload"> for above-the-fold fonts
  • Limit font files: Only load the weights and styles you actually use
  • Test fallbacks: Ensure your fallback fonts look good and are readable

CSS

/* @font-face declaration */
@font-face {
  font-family: 'CustomFont';
  src: url('../assets/fonts/custom-font.woff2') format('woff2'),
       url('../assets/fonts/custom-font.woff') format('woff');
  font-weight: normal;
  font-style: normal;
  font-display: swap;
}

/* Apply custom font with fallbacks */
.custom-font-demo {
  font-family: 'CustomFont', 'Georgia', serif;
}

/* System font stack */
.system-font {
  font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
}

/* Custom font with fallback */
.custom-font {
  font-family: 'CustomFont', 'Georgia', serif;
}

/* Font weight variants */
.weight-100 { font-weight: 100; }
.weight-300 { font-weight: 300; }
.weight-400 { font-weight: 400; }
.weight-500 { font-weight: 500; }
.weight-700 { font-weight: 700; }
.weight-900 { font-weight: 900; }

/* Font demo styling */
.font-demo {
  font-size: var(--fs-lg);
  line-height: 1.6;
  margin: var(--space-4) 0;
  padding: var(--space-4);
  background: var(--font-demo-bg);
  border-radius: var(--radius-1);
  border-left: 4px solid var(--brand);
}

.font-demo h4 {
  color: var(--brand);
  margin-bottom: var(--space-2);
  font-size: var(--fs-sm);
  text-transform: uppercase;
  letter-spacing: 0.05em;
}

Notes

  • @font-face: Declares custom fonts for use in CSS
  • Font formats: WOFF2 (best), WOFF (good), TTF/OTF (fallback)
  • Font-display: Controls how fonts are displayed during loading
  • Fallback fonts: Always provide system font fallbacks
  • Performance: Custom fonts can slow down page loading
  • Preloading: Use <link rel="preload"> for critical fonts
  • Font subsetting: Include only the characters you need
  • Accessibility: Ensure custom fonts meet readability standards
  • Licensing: Make sure you have proper licenses for commercial fonts
