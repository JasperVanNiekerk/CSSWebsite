Fluid Layouts

Learn how to create flexible layouts that adapt to different screen sizes using percentage-based widths, flexbox, and CSS Grid. Fluid layouts are essential for responsive web design.

Live Demo

Flexbox Fluid Layout

This layout uses flexbox with flexible items that grow and shrink based on available space:

Flex Item 1

This item grows and shrinks with the container. It has a minimum width of 200px.

Flex Item 2

All flex items share the available space equally. They wrap to new lines on smaller screens.

Fluid Layout Techniques

  • Percentage widths: width: 50% for relative sizing
  • Flexbox: display: flex with flex: 1 for flexible items
  • CSS Grid: grid-template-columns: 1fr 2fr 1fr for fractional units
  • Auto-fit: repeat(auto-fit, minmax(250px, 1fr)) for responsive grids
  • Max-width: max-width: 1200px to prevent layouts from becoming too wide
  • Min-width: min-width: 200px to prevent items from becoming too narrow

CSS

/* Flexbox fluid layout */
.flex-container {
  display: flex;
  gap: var(--space-4);
  margin: var(--space-4) 0;
  flex-wrap: wrap;
}

.flex-item {
  background: var(--surface);
  padding: var(--space-4);
  border-radius: var(--radius-1);
  border: 1px solid var(--border);
  flex: 1;
  min-width: 200px;
}

/* Percentage-based grid */
.percentage-demo {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  gap: var(--space-4);
  margin: var(--space-4) 0;
}

.percentage-item {
  background: var(--surface);
  padding: var(--space-4);
  border-radius: var(--radius-1);
  border: 1px solid var(--border);
  text-align: center;
}

/* Auto-fit responsive grid */
.responsive-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: var(--space-4);
  margin: var(--space-4) 0;
}

.grid-item {
  background: var(--surface);
  padding: var(--space-4);
  border-radius: var(--radius-1);
  border: 1px solid var(--border);
}

Notes

  • Use relative units: %, fr, em, rem instead of fixed pixels
  • Flexbox: Great for one-dimensional layouts (rows or columns)
  • CSS Grid: Perfect for two-dimensional layouts
  • Auto-fit: Automatically adjusts number of columns
  • Min-width: Prevent items from becoming too narrow
  • Max-width: Prevent layouts from becoming too wide
  • Gap property: Use for consistent spacing between items
  • Test thoroughly: Check on different screen sizes
